# Security Policy

Codex Crash Recovery is local-only. It creates a limited interactive-user Windows Scheduled Task and never opens a network listener.

## Reporting A Vulnerability

Do not include secrets, local paths, session data, logs, or proof-of-concept details in public issues. Report suspected privilege escalation, unintended remote exposure, execution outside the current Windows user, or unsafe recovery behavior privately to the maintainers first. Use GitHub private vulnerability reporting when it is available for this repository; otherwise contact the maintainers through the GitHub organization before public disclosure.

## Maintenance Scope

This project is provided as-is and may be updated intermittently or not at all. Security reports are welcome, but there is no guaranteed response time, remediation commitment, support channel, compatibility guarantee, or maintenance schedule. Users are responsible for reviewing and testing the source before relying on it in unattended workflows.
